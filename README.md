## Install application
Create ` .env ` file with ` APP_URL="localhost" `
```
npm install
```

## Run tests
```
npm run test
```

## Run aplication in developer mode
```
npm run dev
```

## Start aplication
```
npm run start
```

## Stop aplication
```
npm run stop
```
